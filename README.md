# Hyprland Config

- My personal Hyprland setup with custom keybinds, rules, and themes -> This is my backup incase something happens.
- Base config is based on [end4-hyprland](https://github.com/end-4/dots-hyprland)

## Structure

- `hyprland.conf` - Main configuration file
- `custom/` - Custom configs (keybinds, rules, env variables, etc.)
- `hyprland/` - Hyprland-specific modular configs
- `hyprlock/` - Screen lock configuration
- `hypridle.conf` - Idle management
- `shaders/` - Custom shaders
- `monitors.conf` - Monitor setup
- `workspaces.conf` - Workspace configuration

## Quick Start

Copy to `~/.config/hypr/` and reload Hyprland (`Super+Shift+R`).
